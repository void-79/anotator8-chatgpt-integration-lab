Что нужно сделать в Cloudflare

Один раз. После этого URL стабильный (не меняется при перезапуске).

  1. Зайди на dash.cloudflare.com (бесплатный аккаунт).
  2. Добавь свой домен (например mydomain.com) — Cloudflare даст инструкции по смене NS-серверов у регистратора.
  3. Открой one.dash.cloudflare.com → Networks → Tunnels → Create a tunnel → Cloudflared.
  4. Назови тунель (например anotator8-lab), скопируй TUNNEL_TOKEN (длинная строка).
  5. Установи cloudflared: winget install --id Cloudflare.cloudflared (Windows) или brew install cloudflared (Mac/Linux).
  6. Запусти тунель (в отдельном окне терминала): 
    (заполнится автоматически)
